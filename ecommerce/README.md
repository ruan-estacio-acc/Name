# E-commerce — Landing & Docs

O design foi inspirado em https://www.figma.com/design/FzeYCRDV6fN4ONTSHQbpQt/E-commerce-Website-Template--Freebie---Community-?node-id=20-2&m=dev
Este repositório contém a landing page do e-commerce e a documentação interna do projeto (em `docs`).

**Tecnologias usadas**
- React
- Vite
- styled-components
- react-router-dom
- ESLint

**Como instalar**
1. Instale as dependências:

```bash
npm install
```

2. Rodar em desenvolvimento:

```bash
npm run dev
```

3. Build para produção:

```bash
npm run build
```

4. Preview do build:

```bash
npm run preview
```

**Descrição da Landing Page**
- A landing page (rota `/`) é a página principal do site: mostra o header, hero, novidades, promoções e o footer. Serve como ponto de entrada para os usuários navegarem pela loja.

**Sobre a pasta `docs`**
- A pasta `docs` contém a documentação do projeto: exemplos de componentes, estilos e páginas de documentação do design.
- Para visualizar a documentação localmente com o servidor de desenvolvimento rodando, acesse a rota `/docs`.
